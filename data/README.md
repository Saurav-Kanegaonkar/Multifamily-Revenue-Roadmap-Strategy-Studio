# Data Sources

This project uses synthetic but workflow-shaped product data for a multifamily revenue intelligence platform. The data is designed to resemble the inputs a product manager would synthesize before writing a roadmap recommendation.

- `customer_feedback.csv`: 120 customer evidence records across interviews, customer success notes, sales calls, support threads, and beta feedback.
- `product_usage.csv`: 960 telemetry rows covering 120 days across eight product opportunities.

The generated data models realistic product-management structures: pricing recommendation adoption, override behavior, explainability engagement, approval latency, manual exports, stale feed risk, customer persona, portfolio size, and product theme. It does not represent any real customer or company performance.
